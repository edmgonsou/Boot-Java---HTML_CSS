1. Como é possível descobrir qual é a fonte de renderização do navegador?
Resp.: Usado o DevTool do navegador, selecione o elemento HTML e a opção computado.

2. É verdade que quando alteramos um CSS diretamente no DEV TOOL automaticamente alteramos o arquivo CSS?
Resp.: não é verdade, a alteração feita no DEVTOOL é somente visual eo arquivo CSS continua intacto.
